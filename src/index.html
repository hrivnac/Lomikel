Web Service

Customisation
  • Servers.jsp contains the list of Gremlin servets
  • Graphs.jsp contains the list of initial graphs, formed as Gremlin queries
  • bootgreamlin property contains the specific Gremlin query
Options
  • profile for initial calls (null, test, autostart)
  • style for pane setup (null, test, simple)

Command Line

$ java -jar ../lib/Lomikel.exe.jar -h
usage: java -jar Lomikel.exe.jar
 -b,--batch           run in a batch
 -g,--gui             run in a graphical windowile
 -h,--help            show help
 -q,--quiet           minimal direct feedback
 -s,--source    source bsh file (init.bsh is also read)

Bugs, Inconsistencies and ToDos

Bugs
  • Individual Vertex properties overriden by Group properties (vis.js bug).
  • Zoom clustering doesn't work in case of circular edges (vis.js bug).
  • Mail and notification don't work.
  • HBase search with empty results fails.
Inconsistencies
  • Switching 'hierarchical on/off changes the graph params.
ToDos
  • Allow modifications from WS Graph.
  • in WS, give equivalent CLI.
