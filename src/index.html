Bugs, Inconsistencies and ToDos

Bugs
  • Individual Vertex properties overriden by Group properties (vis.js bug).
  • Zoom clustering doesn't work in case of circular edges (vis.js bug).
  • Mail and URL notifications don't work.
  • HBase search with empty results fails.
  • Julian date is not correctly interpreted.
Inconsistencies
  • Switching 'hierarchical on/off changes the graph params.
ToDos
  • Show group of columns in HBase.
  • It should be possible to add graphs into view.
