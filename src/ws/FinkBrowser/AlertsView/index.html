This application displays ZTF/LSST alerts projected on the night sky.

  • Dynamic Sky: zooms to current alerts region.
  • Global Sky: shows the entire sphere.
  • 5-prone stars represent ZTF alerts, 10-prone stars represent LSST alerts.
  • Hover alerts for alert details, click for portal link.
  • Service parameters:
    fetchPeriod=10&fetchStart=24&nAlerts=10&magMax=6
