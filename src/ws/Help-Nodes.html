hbase table

The table panel shows and manipulates data loaded from the graph elemts. It allows sorting according to any column, drawing graphs derived from the table values,...

Numerical variables can be selected for generation of several plots. The Evolution Plot shows time evolution of variables selected in x and y columns). The Scatter Plot shows a 2D (x*y) plot of selected variables. The z column gives the cell value (may be absent), it's reflected by the shown circle size. The Sky Viewb> show the cells in the sky map. x,y columns are ignored. The s column gives the cell value which be be used to group results, if it is not selected, the groups will be created to distinguish different x,y columns, it's reflected by the circle color. All plot objects can be clicked on to show their associated information and actions.
